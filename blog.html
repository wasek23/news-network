What is the difference between var let and const?

  • The 'var' keyword is the oldest way of declaring variables in JavaScript.
  • 'var' and 'let' variables can be reassigned, but 'const' variables can not.
  • 'var' variables are function-scoped(only available within the function), while 'let' and 'const' variables are block-scoped(only available within the block).
  • 'const' variables are immutable while 'let' and 'var' variables are not.

What is the difference between arrow and regular function?

Syntax:

// ES5
var add = function(a, b) {
	return a + b
};

// ES6
let add = (a, b) =>  a + b;
  • The arrow function is introduced in ES6 and is also known as the fat arrow function.
  • In the regular function, we have to use the 'return' keyword to return any value. If we don't return anything then the function will return undefined.
  • If the arrow function contains one expression, we can omit the curly braces, and then the expression will be implicitly returned.
  • In regular functions, 'arguments' keywords can be used to access the arguments of which passed to the function.
  • Arrow functions do not have an 'arguments' binding.
  • However, if we want to access arguments in an arrow function, we can use the rest operator

What is the difference between map, forEach, filter and find?

  • 'map' executes the same code on every element in an array and returns a new array with the updated elements.
  • 'forEach' is used to execute the same code on every element in an array but does not change the array and it returns undefined.
  • 'filter' checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.
  • 'find' checks every element in an array to see if it meets a certain criteria and returns first elements that return truthy for the criteria.

Why use template string?

We can use template strings to write our string exactly how we want it to appear, without all the quotation marks and confusion. Template strings make it very easy to spot a missing space and make our string more readable. And we do not have to use + to concat other strings or variables.
